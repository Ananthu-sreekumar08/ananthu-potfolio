# Portfolio

This is my personal portfolio website built using **React.js**.

## About
It showcases my projects, skills, and experience in a unique **Retro / Game Boy** inspired theme. 

## Features
-   **Interactive Background**: Canvas-based particle system that reacts to mouse/touch.
-   **Retro Design**: Custom "Atomic Purple" aesthetic with pixel-art vibes.
-   **Responsive**: Optimized for both desktop and mobile devices.
-   **Dynamic Content**: Data-driven components for easy updates.

## built With
-   React
-   Vite
-   CSS (Custom Styling)
